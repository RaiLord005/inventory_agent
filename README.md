# inventory_agent
An intelligent, full-stack Inventory Management system that utilizes MySQL for data persistence, Python (Flask) for the backend, and an AI Agent logic layer to optimize stock levels using the Economic Order Quantity (EOQ) formula.

Key Features:
Real-Time Dashboard: View current stock levels, safety thresholds, and lead times.

AI Agent Reasoning: Automatically identifies "Critical" and "Warning" stock levels.

EOQ Optimization: Calculates the most cost-effective reorder quantity to minimize shipping and storage costs.

Compressed Data Processing: Efficiently filters and processes high-priority items using Pandas for faster decision-making.

Interactive Frontend: Dynamic HTML/JavaScript interface with color-coded status alerts.

Tech Stack:
Database: MySQL (Relational Database Management System)

Backend: Python 3.x, Flask

Data Science: Pandas (for data compression and vectorized math)

Frontend: HTML5, CSS3, JavaScript (Fetch API)
